# Gemini Instructions for Workoutes

This file provides specific context and rules for the Gemini AI assistant. Gemini will read this file to understand your coding standards and project context.

## Style Guidelines
- Follow standard Swift naming conventions (camelCase for variables, PascalCase for types).
- Use Swift Concurrency (`async`/`await`) where possible.
- Add clear comments for complex logic.

## Architecture & Frameworks
- Prefer SwiftUI for building user interfaces.
- Use MVVM (Model-View-ViewModel) architecture.
